GoTasks

Google Tasks for iPhone and iPad

Features

  • Synchronize tasks with Google
  • Manage multiple accounts
  • Search
  • Drag and drop to add new tasks in place
  • Double tap to edit tasks in place
  • Notifications of upcoming tasks
  • Different sorting options, sort manually, by date, by title or by status
  • Badges showing the number of tasks (due, overdue, today)
  • Highlight due / overdue tasks
  • Offline mode, edit tasks offline and sync them later
  • Google account data is sent securely over HTTPS
